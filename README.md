* Archaeology Site

An educational website introducing archaeology through selected periods and sites.

* Technologies
- HTML
- CSS

* Description
The site provides a general introduction to archaeology, followed by several main articles and related aside sections.  
It includes examples from:
- Prehistory (Lepenski Vir, Vinƒça)
- Antiquity (Sirmium, Viminacium)
- Middle Ages (Studenica, Resava / Manasija)

Additional features include a map of archaeological sites and an interactive table listing these and additional locations.  
The content is supported by several dozen photographs, six video links, and numerous references to professional and academic websites.

Credits and source links for all used images are provided directly on the website.

* Structure
- HTML and CSS files in the root directory
